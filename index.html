Installation & Setup

Add Tailwind CSS and the Inter font to your project, then copy-paste any component.

<!-- Add to your HTML <head> -->
<script src="https://cdn.tailwindcss.com"></script>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800;900&display=swap" rel="stylesheet">

<!-- Optional: Include CMC tokens as CSS variables -->
<link rel="stylesheet" href="tokens/tokens.css">
// Tailwind config extension
tailwind.config = {
  theme: {
    extend: {
      fontFamily: { sans: ['Inter', 'sans-serif'] },
      colors: {
        'cmc-primary': '#0891B2',
        'cmc-primary-light': '#22D3EE',
        'cmc-primary-dark': '#0E7490',
        'cmc-secondary': '#DC2626',
      }
    }
  }
}

Typography

Inter is the primary typeface. Headlines use extrabold/black weights; body uses regular/medium.

8xl / 96px Display
5xl / 48px Hero Heading
4xl / 36px Section Heading
2xl / 24px Subsection
lg / 18px Large Body Text
base / 16px Regular body text for paragraphs and descriptions.
sm / 14px Small text for cards, navigation, and secondary content.
xs / 12px Caption text for labels, meta info, and fine print.
