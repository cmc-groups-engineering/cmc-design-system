1. Logo Usage

CMC
CMC

Logo Mark: A circular gradient emblem transitioning from red (#DC2626) to blue (#2563EB), representing energy, trust, and innovation.

Wordmark: "CMC" in Inter Bold, tracking-wide, positioned to the right of the mark.

Minimum Size: The logo mark should never be smaller than 24x24px. The full logo (mark + wordmark) should have a minimum width of 80px.

Clear Space: Maintain a minimum clear space equal to the height of the logo mark around all sides of the logo.

Color Usage Rules

Use teal (#0891B2 / #22D3EE) as the primary accent for buttons, links, highlights, card borders, and interactive elements.
Use white (#FFFFFF) for primary text on dark backgrounds and for the main CTA button background.
Use gray scale (#D1D5DB, #9CA3AF, #6B7280) for secondary text, muted content, and labels in decreasing hierarchy.
Never use the red and blue logo colors for UI elements, buttons, or text. They are reserved exclusively for the CMC logo mark.
Never use light gray or white backgrounds as the primary page background. The CMC brand is dark-first.

Typography Rules

Page titles and section headings are always UPPERCASE.
Body text and card titles use sentence case.
Use tight letter-spacing (-0.025em) for display headings and normal/wide for body text.
Never use decorative, serif, or script fonts. Inter is the only approved typeface.
Never go below 12px (0.75rem) for any text. Minimum readable size on screens.

5. Imagery & Photography

Style: Photography should feel authentic, professional, and energetic. Favor candid team shots, action moments, and real workplace environments over staged stock photos.

Treatment: On dark backgrounds, images use a dark overlay gradient (from black/70 via black/30 to black/80) to ensure text readability. Team portrait photos use a grayscale filter that transitions to color on hover.

Aspect Ratios: Hero images are full-width at 16:9 minimum. Card images use a 3:2 ratio. Team portraits are 1:1 (square, displayed in circles).

6. Voice & Tone

Brand Personality

  • Confident: Bold statements, strong headlines, direct messaging.
  • Professional: Clean, organized, data-driven language.
  • Growth-oriented: Focus on scaling, results, and leadership.
  • People-first: Emphasis on team, culture, and development.

Writing Guidelines

  • Use active voice: "We help businesses scale" not "Businesses are helped by us"
  • Keep headlines short and impactful (5-8 words max)
  • Use concrete numbers and metrics when possible
  • Avoid jargon, buzzwords, and overly casual language

7. Do's & Don'ts

DO

Use the full color palette consistently

Stick to the defined color tokens. Use teal for accents, grays for text hierarchy.

Maintain generous whitespace

Use 80-96px between major sections. Let content breathe.

Use uppercase for headings

Page titles and section headings should always be uppercase with tight tracking.

Use pill-shaped buttons for primary CTAs

The rounded-full button style is the signature CTA pattern.

Include hover/transition effects

All interactive elements should have smooth 300ms transitions.

Use image overlays for text readability

Always apply dark gradient overlays on images with text.

DON'T

Use light/white page backgrounds

The CMC brand is dark-first. White is only used for specific sections (services cards).

Use red/blue from the logo as UI colors

Logo colors are reserved for the mark only. Use teal for all UI accents.

Use sharp corners on cards and buttons

Always use rounded corners: 12px for cards, full radius for buttons.

Mix multiple font families

Inter is the only approved typeface. No serif, script, or decorative fonts.

Place text on images without overlays

Always use gradient overlays to ensure text contrast and readability.

Use thin borders or decorations

Keep borders subtle (gray-800). Avoid colorful or thick border treatments.

8. Accessibility Guidelines

All CMC digital properties should meet WCAG 2.1 AA standards at minimum. Here are the key considerations for our dark-themed design.

Color Contrast

White on Black 21:1 (AAA)
Gray-300 on Black 12.6:1 (AAA)
Gray-400 on Black 7.7:1 (AAA)
Cyan-400 on Black 8.6:1 (AAA)
Gray-500 on Black 4.6:1 (AA)

Requirements

All text must meet minimum 4.5:1 contrast ratio (AA) against its background.
Large text (18px+ bold, 24px+ regular) can meet 3:1 (AA for large text).
All interactive elements must have visible focus indicators.
Images must include descriptive alt text.
Navigation must be fully keyboard accessible.
Form inputs must have associated labels and error messages.

Note on Gray-500 (#6B7280)

The gray-500 color meets AA contrast against pure black (4.6:1) but only for text 14px and above. For smaller text or critical information, use gray-400 (#9CA3AF) or lighter. Reserve gray-500 for non-essential decorative text like disclaimers and fine print.
