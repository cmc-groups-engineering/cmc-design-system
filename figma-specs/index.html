8. Complete Token Mapping

Map these tokens directly to Figma styles. Use the tokens.json file with the Figma Tokens plugin for automated import.

Figma Import Instructions

  1. 1 Install the "Figma Tokens" plugin from the Figma community.
  2. 2 Open the plugin and go to Settings > Import tokens from JSON.
  3. 3 Upload tokens/tokens.json from this design system package.
  4. 4 The plugin will create all color styles, text styles, and spacing variables automatically.
  5. 5 Apply styles to components using the reference names (e.g., color.brand.primary).
